Getting Started

Follow these quick steps to get Minicraft+ running on your system.

1

Download Game

Grab the latest executable JAR file from our releases page below.

Download Release
2

Java Requirement

Ensure Java 8+ is installed. Verify your current version here:

java -version
3

Launch Engine

Run the JAR file directly or via this command line for logging:

java -jar minicraft.jar
